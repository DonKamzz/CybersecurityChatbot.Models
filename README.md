# Cybersecurity Awareness Chatbot

**Author:** donkamzz

## Description
A chatbot that educates users about cybersecurity threats including phishing, password safety, and online scams.

## Features
- Voice greeting on startup
- User name memory
- Keyword recognition for cybersecurity topics
- Sentiment detection
- Modern GUI interface

## How to Run
1. Open the project in Visual Studio
2. Press F5 to build and run

## Topics Covered
- Password safety
- Phishing detection
- Scam recognition
- Privacy protection

## Requirements
- Windows OS
- .NET 6.0+
