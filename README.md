# Cybersecurity Awareness Chatbot

A WPF-based chatbot that educates South African citizens about cybersecurity threats including phishing, password safety, and online scams.

## Author
- GitHub: donkamzz
- Student: st10459758

## Project Overview
This chatbot helps users learn about online safety through interactive conversation. It covers key cybersecurity topics relevant to South African internet users.

## Features

### Part 1 Features
- Voice greeting on application startup
- ASCII art logo display
- User name collection and personalization
- Basic keyword responses for cybersecurity topics
- Input validation and error handling
- Colored console output with formatting

### Part 2 Features
- WPF Graphical User Interface
- Modern dark theme design
- Keyword recognition for cybersecurity topics
- Random responses for engaging conversations
- Conversation flow with follow-up handling
- Memory feature to remember user details
- Sentiment detection (worried, curious, frustrated)
- Activity logging

## Topics Covered
- Password Safety (strong passwords, password managers)
- Phishing Detection (spotting fake emails, suspicious links)
- Scam Recognition (common scams in South Africa)
- Privacy Protection (social media privacy, data sharing)
- Two-Factor Authentication
- Malware Prevention

## Requirements
- Windows Operating System
- .NET 6.0 or higher
- Visual Studio 2022 or later

## How to Run the Application

### Option 1: Run from Visual Studio
1. Clone or download the repository
2. Open CybersecurityChatbot.sln in Visual Studio
3. Press F5 to build and run

### Option 2: Run executable
1. Navigate to bin/Debug/net6.0-windows/
2. Double-click CybersecurityChatbot.exe

## How to Use
1. Launch the application
2. Enter your name when prompted
3. Type your cybersecurity questions
4. Try these example questions:
   - "Tell me about passwords"
   - "What is phishing?"
   - "How do I spot a scam?"
   - "Help me with privacy"
5. Type "help" to see all available topics
6. Type "clear" to reset the conversation
